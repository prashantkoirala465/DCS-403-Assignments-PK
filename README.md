## Week 1 - Day 2 Assignment

### Q1: Salary Calculation

**File:** `week1_day2_assignment1.cpp`

Calculates employee total salary with bonus percentage.

- Uses `const` for bonus percentage (15%)
- Demonstrates type conversion and floating-point arithmetic
- Formula: `Total Salary = Base Salary + (Base Salary × Bonus %)`

**Run:**

```bash
g++ week1_day2_assignment1.cpp -o salary && ./salary
```

### Q2: Temperature Conversion

**File:** `week1_day2_assignment2.cpp`

Converts Celsius to Fahrenheit.

- Uses constants for conversion factors (9, 5, 32)
- Formula: `F = (C × 9 / 5) + 32`

**Run:**

```bash
g++ week1_day2_assignment2.cpp -o temp && ./temp
```
